Automated PvP Tab Switch

This addon automatically switches your `TAB` target behavior when you enter PvP instances (arenas / battlegrounds).

Installation

- Copy the folder `AutomatedPvPTabSwitch` into your WoW `Interface\AddOns` directory.
- /reload in-game.

Commands

- `/autopvptab` — shows status
- `/autopvptab testpvp` — force PvP binding
- `/autopvptab testnorm` — force normal binding
